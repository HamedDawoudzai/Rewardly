# Screenshots Needed

Add the following screenshots to this folder:

| Filename | Description |
|----------|-------------|
| `login.png` | Login page |
| `dashboard.png` | Main dashboard (shows RBAC features) |
| `user-management.png` | User management page |
| `transactions.png` | Transactions list/detail page |
| `analytics.png` | Analytics dashboard with charts |
| `events.png` | Events management page |
| `promotions.png` | Promotions list page |
| `dark-mode.png` | Any page in dark mode |

## Tips for Great Screenshots

1. **Resolution**: Use 1280x720 or higher
2. **Clean Data**: Use seeded demo data
3. **Login As**: `manager1` / `pass` to show full features
4. **Browser**: Hide bookmarks bar for cleaner look
5. **Dark Mode**: Toggle for the dark-mode.png screenshot
